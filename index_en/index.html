MMS Report Parser - Software Development Process

This documentation outlines the step-by-step development process of the MMS Report Parser software. The program is designed to read PDF reports generated from the Movie Magic Scheduling software, parse their content, and save the data into an Excel spreadsheet.

PDF reports contain information about each scene in the script. Each scene is presented on a separate page and includes details such as the scene number, filming date, brief scene description, characters present, extras, vehicles, etc. The MMS Report Parser reads all the pages of the report and generates an Excel table where each page is a row, and each detail is in its own column.

The program was built using Python, with the help of ChatGPT for code generation and iteration. This documentation covers the tools used and the complete process from setting up the virtual environment and handling PDF data to formatting the final Excel file and testing the program. If you wish to build a similar tool or further develop this one, this documentation can be helpful.

9. Writing the Code

Example of the PDF parser code:

import pdfplumber

def parse_pdf(pdf_path, logger):
    parsed_pages = []

    try:
        with pdfplumber.open(pdf_path) as pdf:
            for page_num, page in enumerate(pdf.pages, start=1):
                text = page.extract_text()
                if text:
                    lines = text.split('\n')
                    parsed_pages.append({
                        'page_number': page_num,
                        'lines': lines
                    })
                    logger.info(f"Parsed page {page_num}.")
                else:
                    logger.warning(f"No text found on page {page_num}.")
    except Exception as e:
        logger.error(f"Failed to parse PDF: {e}")
        return None

    return parsed_pages
